## Overview

Cloud Extensions expands traditional enterprise infrastructure into modern hybrid cloud environments through cloud services, virtualisation platforms, identity integration, infrastructure automation, and server modernisation.

This section contains reference architectures, governance standards, migration frameworks, implementation guidance, and operational practices supporting cloud adoption and hybrid infrastructure strategies.

### Hybrid First

### Automation
